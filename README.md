Overview

This project is an interactive Retrieval-Augmented Generation (RAG) Chatbot built with the Django framework. The chatbot integrates retrieval mechanisms with generative AI to provide accurate, contextual responses based on indexed knowledge. It is designed for real-world use cases such as document Q&A, knowledge assistant interfaces, and intelligent help systems.

The application supports user authentication (Login/Signup) and provides a simple web UI for real-time conversation with the RAG model.

Features

Built using Django (complete backend and routing)

User Authentication (Login, Signup, Logout)

Chat interface with contextual response generation

Integration with an AI/Vector store retrieval system

Deployed live with Full functionality

Clean and responsive UI

Installation & Setup (Local)

Clone the repository

git clone https://github.com/<your-username>/<your-repo-name>.git
cd <your-repo-name>


Create virtual environment

python3 -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate


Install dependencies

pip install -r requirements.txt


Configure environment variables
Create a .env file and add keys such as:

SECRET_KEY=<your Django secret key>
OPENAI_API_KEY=<your OpenAI key>


Run migrations

python manage.py makemigrations
python manage.py migrate


Start the server

python manage.py runserver


Visit locally
Open http://127.0.0.1:8000
 in your browser.
